# RETRO-MUSIC 🎮

> A cyberpunk-styled web player for streaming video game music from Rainwave.cc

[![License](https://img.shields.io/badge/license-MIT-blue.svg)](LICENSE)
[![Platform](https://img.shields.io/badge/platform-Web-brightgreen.svg)](https://github.com)
[![AirPlay](https://img.shields.io/badge/AirPlay-Supported-yellow.svg)]()

A modern, cyberpunk-themed web radio player streaming video game music, chiptunes, and OC ReMix tracks directly from [Rainwave.cc](https://rainwave.cc/). Features Apple AirPlay support for streaming to external audio devices.

![RETRO-MUSIC Player](https://img.shields.io/badge/RETRO--MUSIC-Player-FF00FF?style=for-the-badge&logo=github)

## ✨ Features

- 🎵 **6 Music Channels**: Game, OC ReMix, Covers, Chiptunes, All, Chill
- 🎮 **Video Game Music**: Stream original game soundtracks, remixes, and chiptunes
- 📡 **Live Streaming**: Real-time radio streams with instant channel switching
- 🍎 **Apple AirPlay**: Stream audio directly to AirPlay-enabled devices
- 🌐 **Cyberpunk UI**: Futuristic dark theme with neon accents
- 📱 **Fully Responsive**: Works on desktop, tablet, and mobile
- 🔊 **Volume Control**: Adjustable volume with mute toggle
- ⌨️ **Keyboard Support**: Quick channel navigation

## 🔧 Installation

```bash
# Clone the repository
git clone https://github.com/yourusername/game-chiptunes.git

# Open in browser
cd game-chiptunes
open index.html
```

Or simply open `index.html` in any modern web browser.
